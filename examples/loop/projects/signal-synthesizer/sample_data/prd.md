# Fieldwork Q1 Roadmap

## Priority 1 — Onboarding Redesign

Rewrite the data import flow (step 3). New CSV validator with inline error messages. Support for common address formats. Estimated: 2 weeks.

## Priority 2 — Team Collaboration Features

Real-time job assignment and status updates across team members. In-app messaging. Notification system for job completions. Team activity feed.

This is a key differentiator vs competitors. Estimated: 4 weeks.

## Priority 3 — Reporting & Export

Exportable PDF and CSV reports. Weekly summary email. Basic revenue dashboard.

## Priority 4 — API & Integrations

Public API for connecting Fieldwork to QuickBooks, Xero, and Zapier. Estimated: 3 weeks.
